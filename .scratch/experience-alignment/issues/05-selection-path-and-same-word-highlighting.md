# 05 — 原位置与同词高亮控制

**What to build:** 让保存词汇高亮区分原始选择位置和整页同词匹配，并在设置中提供独立开关，避免刷新后把所有相同单词都自动高亮。

**Blocked by:** None — can start immediately.

Status: ready-for-agent

- [ ] 设置保留高亮总开关，并新增默认关闭的“高亮相同单词”开关，两个开关变化均立即影响当前页面。
- [ ] 同词高亮关闭时，仅通过保存时记录的 selection path 标记原始选择位置；路径缺失、无效或页面已变化时不得退化为全页词汇匹配。
- [ ] 同词高亮开启时，保留既有的整页匹配、高亮清理和动态内容扫描行为。
