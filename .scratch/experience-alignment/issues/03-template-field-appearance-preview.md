# 03 — 模板字段外观 CSS 与实时预览

**What to build:** 允许每个模板字段快照单独定义 Key CSS 与 Value CSS 声明块，并在外观编辑时实时预览完整当前模板在翻译面板中的显示效果。样式只属于该模板快照，不影响字段定义或其他模板。

**Blocked by:** 02 — 模板字段库与字段快照编排.

Status: ready-for-agent

- [ ] 每个字段快照提供独立的外观编辑入口，可编辑 Key CSS 与 Value CSS 声明块。
- [ ] 编辑界面在 CSS 输入区旁显示使用模拟结果的完整翻译面板预览，无需发起真实翻译请求。
- [ ] 外观修改先保留在模板草稿，只有保存模板才持久化；重置仅清除当前字段快照的两段 CSS。
- [ ] 已保存的 CSS 同时应用到翻译面板中该字段的标签和值，且无效声明不会影响其他字段的渲染。
