# 01 — 模板字段定义与快照模型迁移

**What to build:** 将现有查询模板升级为由可复用的模板字段定义和独立的查询 schema 字段快照组成，并安全迁移现有本地数据。已保存模板中的快照在来源定义被编辑或删除后保持不变；每个模板始终保留至少一个启用字段，系统默认模板继续只读。

**Blocked by:** None — can start immediately.

Status: ready-for-agent

- [ ] 现有模板和活动模板设置可无数据丢失地迁移到字段定义与字段快照模型，并通过数据库重新打开验证。
- [ ] 字段定义的后续编辑或删除不改变既有模板快照；一个模板可包含同一字段定义的多个独立快照。
- [ ] 查询执行、模板校验和默认模板继续以字段快照工作，并拒绝没有启用字段的模板。
