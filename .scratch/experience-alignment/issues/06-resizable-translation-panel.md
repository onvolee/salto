# 06 — 可调整尺寸的翻译面板

**What to build:** 让翻译面板可从右边、底边和右下角拖拽调整尺寸，并在后续划词翻译中恢复最后选择的大小。

**Blocked by:** None — can start immediately.

Status: ready-for-agent

- [ ] 面板最小尺寸保持当前 `360 x 220`，并分别提供右边、底边和右下角调整手柄。
- [ ] 调整时面板左上角位置保持不变，最大宽高始终受当前视口和四周 `8px` 边距限制。
- [ ] 最后选择的尺寸作为全局扩展设置持久化，并在新面板打开和视口变窄时安全裁剪至可见范围。
