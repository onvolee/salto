# OpenDesign 原型规格

本文档描述需要原型的页面和交互，供 OpenDesign 生成 HTML 原型使用。

## 设计系统约束

- **风格**: "The Quiet Marginal Note" — 紧凑、克制、生产力工具风格
- **颜色**: Linear 中性色 + 单一冷紫色主色调 (Linear Violet)
- **排版**: 系统无衬线字体，紧凑尺寸
- **圆角**: 按钮 6px，面板 8px，药丸 999px
- **间距**: xs=4px, sm=8px, md=12px, lg=16px
- **阴影**: 仅用于浮动表面，低模糊度

详细设计规范见 `apps/extension/DESIGN.md`。

## 页面 1: 选项页 — 模板管理

**路径**: `apps/extension/entrypoints/setting.options/`
**尺寸**: 可调整大小的浏览器选项页，最大宽度约 800px

### 布局结构

```
┌─────────────────────────────────────────────────────────────┐
│  Header: "Salto 设置"                                        │
├─────────────────────────────────────────────────────────────┤
│  Tabs: [模板] [设置] [关于]                                   │
├─────────────────────────────────────────────────────────────┤
│                                                              │
│  ┌─────────────────────────────────────────────────────────┐│
│  │ 模板列表                                                 ││
│  │ ┌─────────────────────────────────────────────────────┐││
│  │ │ ● 默认模板 (默认)                                    │││
│  │ │   翻译 + 音标 + 词性 + 释义                          │││
│  │ │   [编辑] [复制] [删除]                               │││
│  │ └─────────────────────────────────────────────────────┘││
│  │ ┌─────────────────────────────────────────────────────┐││
│  │ │ 词汇学习模板                                         │││
│  │ │   翻译 + 例句 + 同义词                              │││
│  │ │   [编辑] [复制] [设为默认] [删除]                    │││
│  │ └─────────────────────────────────────────────────────┘││
│  │                                                         ││
│  │ [+ 新建模板]                                            ││
│  └─────────────────────────────────────────────────────────┘│
│                                                              │
└─────────────────────────────────────────────────────────────┘
```

### 组件层次

1. **Header**
   - 标题: "Salto 设置" (600, 0.875rem)
   
2. **设置导航**
   - 沿用现有设置侧边栏分区：通用 / 划词翻译 / 翻译源 / 词汇 / AI 服务
   - 活动分区使用现有 sidebar active 状态，不新增独立 Tabs 页面

3. **模板列表**
   - 每个模板卡片:
     - 名称 (600, 0.75rem)
     - 字段摘要 (400, 0.75rem, muted-foreground)
     - 默认标记: 小紫色圆点 + "默认" 标签
     - 操作按钮: 编辑 / 复制 / 删除 / 设为默认
     - 最后一个模板时删除按钮禁用

4. **新建模板按钮**
   - Ghost 样式，左对齐
   - 点击打开模板编辑抽屉

### 交互状态

- **默认**: 显示模板列表
- **Hover**: 卡片背景变为 muted，按钮显示
- **Focus**: 键盘焦点显示 Focus Violet 环
- **Disabled**: 删除最后一个模板时禁用删除按钮
- **Loading**: 模板列表加载时显示骨架屏

## 页面 2: 选项页 — 模板编辑

**触发**: 点击模板卡片的"编辑"或"新建模板"

### 布局结构

```
┌─────────────────────────────────────────────────────────────┐
│  ← 返回模板列表                                              │
├─────────────────────────────────────────────────────────────┤
│  模板名称: [________________]                                │
├─────────────────────────────────────────────────────────────┤
│  字段列表                                                     │
│  ┌─────────────────────────────────────────────────────────┐│
│  │ ≡ 翻译                                                   ││
│  │   来源: [LLM ▼]  类型: [text ▼]  启用: [●]              ││
│  │   指令: [将选中文本翻译成{{targetLanguage}}__________]    ││
│  │   [插入变量 ▼]                                          ││
│  │   [删除字段]                                            ││
│  └─────────────────────────────────────────────────────────┘│
│  ┌─────────────────────────────────────────────────────────┐│
│  │ ≡ 音标                                                   ││
│  │   来源: [字典 ▼]  字段: [phonetic ▼]  启用: [●]          ││
│  │   [删除字段]                                            ││
│  └─────────────────────────────────────────────────────────┘│
│  ┌─────────────────────────────────────────────────────────┐│
│  │ ≡ 释义                                                   ││
│  │   来源: [字典 ▼]  字段: [meaning ▼]  启用: [●]           ││
│  │   [删除字段]                                            ││
│  └─────────────────────────────────────────────────────────┘│
│                                                              │
│  [+ 添加字段]                                                │
├─────────────────────────────────────────────────────────────┤
│  [保存] [取消]                                               │
└─────────────────────────────────────────────────────────────┘
```

### 组件层次

1. **返回链接**
   - Ghost 按钮，左箭头图标 + "返回模板列表"

2. **模板名称输入**
   - Label: "模板名称"
   - Input: 文本输入，必填

3. **字段列表** (可拖拽排序)
   - 每个字段卡片:
     - 拖拽手柄: 左侧六点图标 (≡)
     - 字段名称: 可编辑文本输入
     - 来源选择: Select 组件 (LLM / 字典)
     - 类型/字段选择: 
       - LLM 来源时: 类型选择 (text / list)
       - 字典来源时: 字段选择 (phonetic / partOfSpeech / meaning / synonyms / wordForms)
     - 启用开关: Switch 组件
     - LLM 指令输入 (仅 LLM 来源):
       - Textarea: 多行文本输入
       - 变量插入按钮: 下拉菜单，列出可用变量
       - 变量格式: `{{variableName}}`
     - 删除字段按钮: Ghost 按钮，红色文字

4. **添加字段按钮**
   - Ghost 样式，左对齐
   - 点击添加新字段到列表底部

5. **操作按钮**
   - 保存: Primary 按钮
   - 取消: Ghost 按钮

### 交互状态

- **拖拽中**: 字段卡片抬起，显示阴影，其他字段让出空间
- **变量菜单打开**: 下拉显示可用变量列表
- **未知变量警告**: 保存时如果指令中有未知变量，显示黄色警告横幅
- **必填错误**: 模板名称为空时显示红色边框和错误消息
- **来源切换**: 切换来源时清空对应字段内容，显示确认对话框

### 变量插入菜单

```
┌─────────────────────────────┐
│ selection                   │
│ sentence                    │
│ paragraphs                  │
│ targetLanguage              │
│ webTitle                    │
│ webUrl                      │
│ webContent                  │
└─────────────────────────────┘
```

## 页面 3: 选项页 — 设置

**路径**: `apps/extension/entrypoints/setting.options/` (设置侧边栏)

### 布局结构

```
┌─────────────────────────────────────────────────────────────┐
│  Header: "Salto 设置"                                        │
├─────────────────────────────────────────────────────────────┤
│  Tabs: [模板] [设置] [关于]                                   │
├─────────────────────────────────────────────────────────────┤
│                                                              │
│  翻译设置                                                     │
│  ┌─────────────────────────────────────────────────────────┐│
│  │ 目标语言: [中文 ▼]                                       ││
│  │ 高亮已保存词汇: [●]                                      ││
│  └─────────────────────────────────────────────────────────┘│
│                                                              │
│  LLM 配置                                                     │
│  ┌─────────────────────────────────────────────────────────┐│
│  │ 提供者: [OpenAI ▼]                                       ││
│  │ 模型: [gpt-4o-mini ▼]                                    ││
│  │ API 密钥: [已配置] [替换]                                 ││
│  │                                                         ││
│  │ ⚠️ 需要 exact-origin 权限                                ││
│  │ [授予权限]                                               ││
│  └─────────────────────────────────────────────────────────┘│
│                                                              │
│  外观                                                         │
│  ┌─────────────────────────────────────────────────────────┐│
│  │ 主题: (●) 跟随系统  ( ) 浅色  ( ) 深色                    ││
│  └─────────────────────────────────────────────────────────┘│
│                                                              │
└─────────────────────────────────────────────────────────────┘
```

### 组件层次

1. **翻译设置**
   - 查询模板: Select 下拉菜单，选择并保存 `activeQueryTemplateId`
   - 目标语言: Select 下拉菜单
   - 高亮开关: Switch 组件 + Label

2. **LLM 配置**
   - 提供者选择: Select 下拉菜单
   - 模型选择: Select 下拉菜单 (根据提供者动态变化)
   - API 密钥:
     - 已配置时: 显示 "已配置" + 替换按钮
     - 未配置时: 显示密码输入框 + 保存按钮
   - 权限提示 (如需要):
     - 警告图标 + 说明文字
     - 授予权限按钮

3. **外观设置**
   - 主题选择: 三个单选按钮 (跟随系统 / 浅色 / 深色)

### 交互状态

- **API 密钥输入**: 密码类型，可切换显示
- **权限授予中**: 按钮显示加载状态
- **权限已授予**: 隐藏警告提示
- **保存成功**: 显示短暂的绿色勾选提示

## 页面 4: 翻译面板

**路径**: `entrypoints/content.tsx` (Shadow DOM 内)
**尺寸**: 360px × 220px (最大)

### 布局结构

```
┌────────────────────────────────────┐
│ ≡ 默认模板                 [×]      │  ← 可拖拽头部
├────────────────────────────────────┤
│                                    │
│  翻译                               │
│  将选中文本翻译成中文               │
│  ┌──────────────────────────────┐ │
│  │ 这是一个翻译结果示例          │ │
│  └──────────────────────────────┘ │
│                                    │
│  音标                               │
│  /ˈɛksæmpəl/                       │
│                                    │
│  释义                               │
│  一个代表性的人、事、物             │
│                                    │
├────────────────────────────────────┤
│ [保存词汇]          [复制] [朗读]  │  ← 底部工具栏
└────────────────────────────────────┘
```

### 组件层次

1. **头部**
   - 拖拽手柄: 左侧六点图标 (≡)
   - 模板名称: 只读文本，显示当前 settings 选择的活动模板
   - 关闭按钮: 右侧 × 图标

2. **内容区**
   - 字段列表 (根据模板动态渲染):
     - 字段标签 (600, 0.75rem)
     - 字段内容 (400, 0.75rem)
     - 加载状态: 骨架屏
     - 错误状态: 红色文字 + 重试按钮
     - 空状态: "无结果" 灰色文字

3. **底部工具栏**
   - 保存词汇: Primary 按钮
   - 复制: Ghost 按钮，复制图标
   - 朗读: Ghost 按钮，扬声器图标

### 交互状态

- **活动模板**: 显示 Settings 中已保存的活动模板名称；面板内没有模板选择器
- **加载中**: 字段内容显示骨架屏
- **错误**: 字段内容显示红色错误消息 + 重试按钮
- **保存成功**: 按钮变为绿色勾选，2秒后恢复
- **保存失败**: 显示错误提示
- **拖拽中**: 面板跟随鼠标移动
- **关闭**: 按 Escape 或点击 × 关闭面板

## 页面 5: 高亮样式

**位置**: 网页内容中

### 样式规格

```
高亮文本:
- 背景: rgba(109, 120, 213, 0.15)  /* Linear Violet 15% 透明度 */
- 下边框: 2px solid rgba(109, 120, 213, 0.4)
- 圆角: 2px
- 过渡: background-color 150ms ease

悬停状态:
- 背景: rgba(109, 120, 213, 0.25)
- 光标: pointer

Tooltip:
- 位置: 高亮文本上方
- 内容: 词汇信息 (翻译、音标等)
- 样式: 与翻译面板一致
```

### 交互

- **悬停**: 显示 tooltip，显示词汇简要信息
- **点击**: 打开翻译面板，显示完整信息
- **禁用高亮**: 移除所有高亮，恢复原始文本

## 组件清单

需要使用的现有组件 (见 `apps/extension/src/components/ui/`):

- `button` — Primary / Ghost / Outline 变体
- `input` — 文本输入
- `textarea` — 多行文本输入
- `select` — 下拉选择
- `switch` — 开关
- `label` — 表单标签
- `separator` — 分隔线
- `tooltip` — 工具提示
- `spinner` — 加载指示器
- `skeleton` — 骨架屏
- `badge` — 标签 (如 "默认")
- `alert` — 警告提示

需要新增的组件:

- `draggable-list` — 可拖拽排序列表 (使用 @dnd-kit)
- `variable-menu` — 变量插入下拉菜单
- `template-card` — 模板列表卡片
- `field-editor` — 字段编辑卡片

## 技术决策

### 已确定

| 功能 | 技术选型 |
|------|----------|
| 表单处理 | `@tanstack/react-form` + `zod` |
| 拖拽排序 | `@dnd-kit/core` + `@dnd-kit/sortable` |
| UI 组件 | shadcn + Base UI |
| 样式 | Tailwind CSS v4 + 语义变量 |
| 图标 | `@hugeicons/react` + `@hugeicons/core-free-icons` |

### 需要安装的 shadcn 组件

- `dialog` — 模板删除确认
- `dropdown-menu` — 模板操作菜单
- `checkbox` — 字段启用/禁用
- `popover` — 变量插入面板
