# 09 — 翻译面板模板切换

**What to build:** 翻译面板加载默认模板，支持面板内模板切换（不改变全局默认），模板编辑后下次翻译立即生效，模板删除或损坏时安全恢复。

**Blocked by:** 05 — 模板持久化和仓库操作, 06 — 模板字段编辑和排序

**Status:** ready-for-agent

- [ ] 加载默认模板：面板打开时加载默认模板
- [ ] 面板内模板切换：支持面板内模板切换而不改变全局默认
- [ ] 切换行为定义：定义切换是立即触发执行还是通过显式运行操作，保持一致并测试
- [ ] 编辑立即生效：模板编辑应用于后续运行，无需扩展重新加载
- [ ] 删除/损坏恢复：当选定/默认模板被删除或损坏时安全恢复
- [ ] 订阅设置变更：扩展表面订阅相关设置变更并在卸载时移除监听器
- [ ] 运行时测试：证明模板编辑改变下一次翻译请求
- [ ] 面板测试：覆盖模板切换、编辑生效、恢复行为
