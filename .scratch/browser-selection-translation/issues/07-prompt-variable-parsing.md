# 07 — 提示词变量解析和验证

**What to build:** 用户可以在 LLM 指令中插入和编辑提示词变量，编辑器提供变量插入菜单，保存时验证未知变量但不阻止保存。

**Blocked by:** 05 — 模板持久化和仓库操作

**Status:** ready-for-agent

- [ ] 变量插入菜单：为每个支持的 PromptContext 变量（selection、sentence、paragraphs、targetLanguage、webTitle、webUrl、webContent）提供插入菜单
- [ ] shadcn 组件安装：安装 `popover`（变量插入面板）
- [ ] 变量解析器：使用运行时相同的解析器解析手动输入的变量
- [ ] 未知变量警告：保存时警告未知变量但不阻止保存
- [ ] 区分未知和不可用：区分未知变量和暂时不可用的上下文值
- [ ] 预览或验证摘要：显示简洁的预览或验证摘要，不在每次编辑时发送提供者请求
- [ ] 解析器测试：覆盖有效、重复、相邻、未知和畸形变量
- [ ] 验证测试：覆盖未知变量警告、保存阻止行为、预览摘要
