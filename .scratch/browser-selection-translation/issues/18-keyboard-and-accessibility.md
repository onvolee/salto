# 18 — 键盘快捷键和无障碍

**What to build:** 在清单中定义和注册扩展键盘命令，验证逻辑焦点进入、包含和恢复，为每个图标操作提供可访问名称和工具提示，验证键盘优先的模板、设置、重试、保存、面板工作流，验证 WCAG AA 对比度。

**Blocked by:** 09 — 翻译面板模板切换, 17 — 保存后高亮集成

**Status:** ready-for-agent

- [ ] 键盘命令注册：在清单中定义和注册扩展键盘命令
- [ ] 焦点管理：验证逻辑焦点进入、适当时的包含和面板关闭后的恢复
- [ ] 可访问名称：为每个图标操作提供可访问名称和工具提示（需要时）
- [ ] 键盘优先工作流：验证仅键盘的模板、设置、重试、保存和面板工作流
- [ ] WCAG AA 对比度：验证浅色、深色和系统模式下的 WCAG AA 对比度
- [ ] 非颜色状态：避免仅颜色表示保存、待处理、失败和就绪状态
- [ ] 减少动画：尊重减少动画并避免移动布局的动画
- [ ] 缩放和窄视口：验证缩放和窄视口文本不重叠或逃出控件
- [ ] 无障碍测试：覆盖标签、焦点顺序、键盘重排、错误关联和对比度关键状态
