# 14 — 纯匹配引擎

**What to build:** 纯匹配引擎实现大小写不敏感英文匹配、完整词边界、最长术语优先（重叠时）、Unicode 和撇号行为，纯匹配计算不涉及 DOM 变更。

**Blocked by:** 04（已完成）

**Status:** ready-for-agent

- [ ] 大小写不敏感匹配：大小写不敏感英文匹配
- [ ] 完整词边界：完整词边界匹配
- [ ] 无词形还原：无词形还原或词形匹配
- [ ] 无跨节点匹配：无跨 DOM 文本节点匹配
- [ ] 最长术语优先：当保存的短语在一个文本节点内与保存的词重叠时，优先最长有效术语
- [ ] Unicode 和撇号行为：显式定义 Unicode 和撇号行为（英文边界）
- [ ] 纯匹配计算：保持纯匹配计算与 DOM 变更分离
- [ ] 归一化和去重：在构造匹配器前归一化和去重保存的术语
- [ ] 正则转义：在创建任何正则表达式前转义所有术语文本
- [ ] 匹配器测试：覆盖空、大、重叠、混合大小写、标点、特殊字符术语集
- [ ] 匹配规则测试：覆盖所有匹配规则
