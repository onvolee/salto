# 06 — 模板字段编辑和排序

**What to build:** 用户可以编辑模板字段的标签、来源、类型、启用状态和顺序，编辑 LLM 指令或选择字典字段，并通过键盘可访问的控件重排字段。

**Blocked by:** 05 — 模板持久化和仓库操作

**Status:** ready-for-agent

- [ ] 字段标签编辑：编辑字段显示标签
- [ ] 字段来源切换：在 LLM 和字典之间切换字段来源
- [ ] 字段类型编辑：编辑 LLM 字段的类型（text/list）
- [ ] LLM 指令编辑：编辑 LLM 字段的指令文本
- [ ] 字典字段选择：选择字典字段（phonetic/partOfSpeech/meaning/synonyms/wordForms），类型从冻结映射派生
- [ ] 字段启用/禁用：切换字段的启用状态
- [ ] shadcn 组件安装：安装 `checkbox`（字段启用/禁用切换）
- [ ] 字段重排：使用 @dnd-kit/core + @dnd-kit/sortable 实现键盘可访问的拖拽重排
- [ ] 顺序值归一化：编辑后事务性地归一化顺序值
- [ ] 防御性保留：在未来迁移中防御性地保留未知提供者或字段数据，而不是静默丢弃
- [ ] 切换验证：切换字段来源前需要有效的 LLM 指令或字典字段选择
- [ ] UI 测试：覆盖字段编辑、重排、来源切换、类型验证
- [ ] 表单验证：使用 @tanstack/react-form + zod 处理字段编辑表单
