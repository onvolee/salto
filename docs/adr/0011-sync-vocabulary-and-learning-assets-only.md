# Sync vocabulary and learning assets only

Future Salto sync will include vocabulary and learning records: vocabulary items, vocabulary fields, vocabulary contexts, learning cards, learning states, and review logs. Browser-extension workspace data such as query templates, LLM configuration, dictionary provider configuration, highlight settings, translation caches, enrichment jobs, and UI state will remain local to the extension. This lets mobile clients consume stable study data without needing to understand extension-specific prompting, scraping, highlighting, or background job mechanics.
